### Project Setup

1. Clone the project

```bash
git clone https://github.com/skycoder026/mediusware.git

Please use the branch dev.0.0.1
```

2. Run

```bash
composer install && npm install && npm run build
```
3. Create Database and configure

4. run migration and seeder

```bash
php artisan migrate --seed
```
